## 作成した6つのアプリ

後半の6個については、「どんな思想で作ったか」「どんな壁にぶつかったか」に焦点を当ててまとめました。
1〜3個目は通常通り開発し、残る4〜6個目は「1時間でどこまで作れるか」というタイムアタックで構築しています。

### 1. [わりかんの相棒 (Warikan App)](https://warikan-bugyou.vercel.app)

イベントごとの精算記録から、請求メッセージの自動生成までが完結するシンプルな割り勘アプリ。

![](/images/warikan.png)

| 項目 | 内容 |
| :--- | :--- |
| **思想** | 個人的に毎月5~10回ほどの飲み会で幹事をしており、管理が非常に煩雑。取りこぼしも多いため、**自身の痛み**から開発。（全部奢れるくらいの大きな男になった暁には、このアプリを削除するのが目標です） |
| **工夫** | 若手・役職者などへの**「傾斜計算」**を即座に対応。また、複数の振込先口座入りの請求文面を自動生成し、支払側のハードルを下げる工夫を行いました。 |
| **壁** | 細かな調整はあったものの、割と順風満帆に進行し、大きな壁はありませんでした！ |
| **展望** | レシート写真のOCR読み取りによる自動割り振りや、LINE/Slackのメッセージ内容からの自動精算連携。 |

### 2. [習慣資産 (Habit Asset)](https://habit-asset.vercel.app)

「失敗も資産にする」をコンセプトにした、継続のためのトラッキングアプリ。

![](/images/habit.png)

| 項目 | 内容 |
| :--- | :--- |
| **思想** | 禁酒アプリが続かない原体験から打破。1日の失敗で全てが台無しになる絶望感をなくし、負の習慣を断つだけでなく、筋トレなどの正の習慣も同時に積み上げる思想。 |
| **工夫** | 毎回入力する手間を排した**「自動積み上げ」**と、失敗した日を「全てリセット」ではなく**「その日だけ計上しない」**方式にし、継続の糸を切らさないUI。 |
| **壁** | **出来が微妙すぎる。** 思想的には面白いと思っているのですが、形に落とし込むのが難しく、世にある禁酒・禁煙アプリがなぜ「あえて特化しているのか」が痛いほど分かりました。 |
| **展望** | サービスを閉じるかもしれないです……。特化型アプリの強さを学ぶ良い機会になりました。 |

## 4日間で気づいた、正直なこと3つ

限界開発をやってみて、技術的な話よりも「自分のこと」がよく見えてきました。

### 1. AIを待つ時間より、自分が詰まる時間のほうが長い

AIに指示してアプリが上がってくるのは本当に早いです。
でも「これでいいか？」「ここ直す？捨てる？」って考える時間、これが思ったよりデカかった。
作るのが速い分、**判断を求められる回数も圧倒的に増える**んですよね。
今後PdMとして一番大事なのは「判断力」だなと。で、その判断力を秒で発揮・磨き上げるためには、思考や仮説検証の経験を日頃から如何に多く積み上げるかだな〜と感じています。

### 2. ドメインわからない領域でAIに作らせると、クオリティの判断もできない

仮想通貨のアプリを作ったんですが、完成したときに「これ、教材として正しいの？」が全くわかりませんでした。
自分が初心者なので当然なんですが、これが一番こわいと思った。
AIが出してきた「それっぽいもの」を世に出してしまうリスク。
これはまるで、**「アミウダケ」を奇跡の薬だと信じてヒルルクに飲ませたチョッパー**と同じです。
万能薬のマーク（ドクロ）だと信じていたものが実は猛毒であったように、受け取り手（PdM）にそれを見極めるドメイン知識がなければ、知らぬ間に毒のあるプロダクトを世に放つことになります。
**「何を作るか」だけでなく「出来上がったものが正しいか確認できるか」もセットで問われる**と実感しました。

### 3. 「企画書を3日かけて作る」よりも「とりあえず動かす」ほうが断然早い

後半の3つは、実質アイデアをAIに投げてそのままデプロイしました。
それでも一応「もの」になって世に出る。
企画の議論でウンウン消耗していた時間を、そのままプロトタイプ作りに使えばいい。
**走りながら考える、の解像度が完全に変わりました。**

## 余録：限界開発がもたらした、マインドの劇的な変化

技術的な知見以上に大きかったのは、**開発を通じて自分自身のマインドや熱量が劇的に変化したこと**です。

*   **興奮状態の常態化（テストステロン値の向上）**
    アイデアが数十分で形になる体験は、もはや麻薬です。「次は誰と（何を）戦おうか」と、まるでヒソカのように常にワクワク・興奮する機会が多くなり、明らかにテストステロン値が跳ね上がった気がします。
*   **「偉い人」への提案が全く怖くなくなった**
    以前はメッセージの文面や分厚い提案資料の作成にクソほど時間をかけていました。しかし今回、「足りなくても（未完成でも）まずは提案し、失敗したら修正すればいい」という精神が完全に根付きました。結果的にあらゆるタスクの初速が圧倒的に上がり、それがものすごい自信に繋がっています。
*   **「1時間でいくつリリースできるか？」という狂気のタイムアタック**
    実際に3つ（仮想通貨ノック、メンズメイク、第一印象スター）を1時間でデプロイできました。冷静に考えてえぐくないですか？
